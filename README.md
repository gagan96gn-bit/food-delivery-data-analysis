# Food Delivery Data Analysis and Preprocessing

Exploratory data analysis and preprocessing on a food delivery order dataset, covering data quality, distributions, and business-relevant insights.

## Overview
Cleans and analyzes food delivery order data (order details, cuisine type, cost, preparation time, delivery time) to answer business questions about restaurant performance, cuisine popularity, and delivery efficiency.

## Approach
- Data structure exploration: column types, qualitative/quantitative classification
- Data cleaning: missing values, data mismatches, duplicate detection
- Outlier detection on cost, food preparation time, and delivery time using box plots and z-scores
- Univariate distribution analysis across all variables
- Business questions answered: top restaurants by order volume, most popular cuisine on weekends, mean delivery time

## Tools
Python, pandas, NumPy, matplotlib, seaborn

## How to run
1. Open the notebook (Colab or local Jupyter)
2. Load the food delivery dataset
3. Run cells sequentially — exploration, cleaning, EDA, question-by-question analysis
